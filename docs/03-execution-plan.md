# Execution Plan (6 Weeks)

## Week 1 - Discovery and Content Inventory
- Finalize goals, KPIs, and stakeholders
- Audit all current page copy and media
- Define voice, capitalization, and style rules
- Finalize sitemap and page templates

Deliverables:
- Final sitemap
- Messaging framework
- Content inventory matrix

## Week 2 - UX/Wireframes
- Build desktop and mobile wireframes for:
  - Home
  - Services overview
  - Service detail template
  - About
  - Careers
  - Contact
- Validate CTA placement and trust blocks

Deliverables:
- Approved wireframes
- Component list

## Week 3 - Visual Design and UI System
- Create modern visual direction aligned to industry credibility
- Build type scale, color system, spacing, and components
- Review accessibility contrast and interaction clarity

Deliverables:
- High fidelity designs
- UI component specs

## Week 4 - Development (Core)
- Set up Next.js project and routing
- Build reusable layout/components
- Implement page templates and CMS integration
- Implement SEO metadata and schema basics

Deliverables:
- Functional staging build
- CMS content model

## Week 5 - Content Migration and QA
- Migrate copy and optimized media
- Set up forms, email notifications, and analytics events
- QA on desktop/mobile, cross-browser, and accessibility checks

Deliverables:
- Content-complete staging
- QA issue log and fixes

## Week 6 - Launch and Stabilization
- Configure redirects and verify canonical setup
- Final production build and smoke tests
- Deploy and monitor
- 2-week post-launch optimization backlog

Deliverables:
- Live site
- Redirect map
- Post-launch performance report

## Technical QA Checklist
- All primary pages have unique titles and descriptions
- Correct html lang and content-language headers
- Open Graph and Twitter cards configured
- Sitemap and robots.txt validated
- Form success/error states tested
- Core Web Vitals baseline recorded
- 404 and redirect behavior verified

## Measurement Plan
Track these events:
- form_submit_contact
- click_phone_header
- click_email_contact
- click_cta_service_detail
- careers_apply_start
- careers_apply_submit

Create dashboards for:
- Sessions by source
- Landing page conversion rate
- Service page assisted conversions
- Form completion rate
